# smartDataQuerying MVP 验收清单

## 后端

- [ ] `POST /api/auth/login` 可登录，未登录接口返回 401。
- [ ] 数据源密码不会明文入库。
- [ ] MySQL 数据源可连接并同步表结构。
- [ ] PostgreSQL 数据源可连接并同步表结构。
- [ ] 表/字段备注、启用状态、敏感标记可更新。
- [ ] 业务术语可新增、修改、删除。
- [ ] 业务术语可维护分类，并在问数时按分类优先召回。
- [ ] SQL 示例可新增、修改、删除。
- [ ] LLM 配置可保存并测试。
- [ ] `/api/chat/ask` 可返回 SQL、解释、列、行、耗时和审计 ID。
- [ ] 危险 SQL 被拒绝且写入审计。
- [ ] Excel `.xlsx` 可导入系统库并自动注册为 `Excel Imports` 数据源。
- [ ] 删除 Excel 导入记录时，物理表和元数据一起删除。

## 前端

- [ ] 登录页可用。
- [ ] 数据源页可新增、测试、同步、查看 schema。
- [ ] 术语库页可新增术语。
- [ ] 术语库页可选择/输入分类、编辑分类、按分类筛选。
- [ ] SQL 示例页可新增示例。
- [ ] 模型设置页可保存和测试配置。
- [ ] 问数页可选择数据源并提问。
- [ ] 审计页可查看最近 100 条查询。
- [ ] 数据源页可上传 Excel，并查看 Excel 导入记录。

## 手动问数样例

准备 10 个真实业务问题，逐条记录：

- 问题文本
- 生成 SQL
- 是否执行成功
- 业务口径是否正确
- 是否需要补充术语或 SQL 示例

首版验收线：10 个问题中至少 7 个生成可执行且口径可接受的 SQL。
